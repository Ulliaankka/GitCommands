Підключення аку: git config --global user.name blabla git config --global user.email blabal@bla Команди: git init - створити новий лок репозиторій git status - перевірка стану git add - додаємо один файл git add . - додаємо всі файли git commit - переносимо дані в репозиторій git log - дивитись інфу про коміти git checkout - git branch - глянути на якій зараз гілці git push git pull git merge git clone - скачує ост версію проекту git revert git reset git rebase git cherry-pick git stash
